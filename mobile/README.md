# Мобильная оболочка MatchMe

В каталоге **`matchme-android/`** — отдельный небольшой проект **Capacitor** для **Android**: полноэкранный **WebView**, который открывает задеплоенный **Next.js** (тот же фронт, что в браузере). Логика и API остаются на сервере; переустанавливать бизнес-логику в Kotlin не нужно.

Сборка и запуск: **`matchme-android/README.md`**.
